Вячеслав Логунов

Родился в 1978 году. Окончил Московский Институт Электронной Техники (МГИЭТ ТУ) по специальности инженер электронной техники.

Какое-то непродолжительное время (год?) работал в сфере Code Reverse Engineering, изучал системы защиты от копирования

Проходил практику в НИИ Микроприборов в Зеленограде, а также в компании TrustWorks B.V., занимавшейся разработкой решений на базе протоколов IPSEC и IKE.

Работал в компании ZELAX в отделе программного обеспечения, где занимался разработкой системного ПО для IP-маршрутизаторов. Реализовывал поддержку IPSec, NAT, Telnet, SSH, фильтрацию IP-трафика, Multilink PPP, драйверы для Ethernet и ATM и многое другое.

Знаком с архитектурами PowerPC, Xtensa (ESP32), RISC-V. Разрабатывал драйверы для Linux (система защиты данных на диске), разрабатывал Board Support Package (BSP) для RTEMS (для маршрутизаторов).

Интересы: электроника, программирование, scuba дайвинг, естественные науки, коты и кошки
