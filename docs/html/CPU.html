⇮SETTING UP CPU FREQUENCY

ESP32 CPU supports frequency change (on classic ESP32 main CPU frequency may affect APB bus frequency also) in runtime:

cpu FREQ

FREQ is the new CPU frequency in MHz (e.g. 240)

ESP32 supports 240, 160, 80 MHz frequencies and also, depending on Xtal frequency, Xtal / 4 and/or Xtal / 2 frequencies can be available. List of available (supported) frequencies is displayed on attempt to set unsupported frequency: 

  esp32#>cpu 1
  % 1 MHz is unsupported frequency
  % Supported frequencies are: 240, 160, 120, 80, 40, 20 and 10 MHz
  % Invalid 1st argument "1" ("? cpu" for help)

⇮DISPLAYING CPU INFORMATION

show cpuid

Displays hardware & software information:


Fig. 1: Output of "show cpuid" command

Output is split into 3 sections: Hardware, Firmware and Last boot. First section contains CPUID, CPU frequency, xtal frequency, APB bus frequency and CPU core temperature.

Second section provides version information on various components used to build the firmware/sketch: ArdunoCore version (mismatched versions are common source of a headache), the ESP-IDF version and ESP32Shell own version number

Third section contains output of "uptime" command: uptime and reason for last reboot.
